    of( name ) click to toggle source

    observatory from a name (should be in Locations). Sphere::Observatory::Locations.keys has the list of the names.

    # File lib/sphere/observatory.rb, line 38
def self::of( name )
  unless Locations.has_key?( name ) then
    raise RuntimeError, %Q|No location with the name "#{name}" found.|
  end
  new( Locations[name][0], Locations[name][1], name )
end
